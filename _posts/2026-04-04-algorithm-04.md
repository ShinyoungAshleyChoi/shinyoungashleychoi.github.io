- 이론 정리
    - 그래프: 정점과 간선으로 이루어진 구조
        - 인접리스트: 노드의 개수만큼 공간을 확보한 뒤, 각 노드를 기준으로 연결된 노드를 저장(빈도 90%)
            - 어떤 노드와 연결된 노드를 가장 쉽게 접근 → 사용 빈도 가장 높음
            - 다익스트라
        - 인접행렬: 노드 개수 * 노드 개수 만큼의 행렬(공간)을 만둔 후에 간선을 표현 (빈도 20%)
            - 어떤 노드 A에서 노드 B로 가는 간선이 있는지를 빠르게 확인할 때 유용
            - 플로이드-워셜 알고리즘
        - 간선리스트: 간선의 정보만(노드 2개로 표현) 표현하는 방식 (빈도 10%)
            - 간선을 기준으로 그래프를 해석할 때 유용
            - 그러나 노드를 찾을 때는 전체 그래프를 봐야하기 때문에 불편
            - 벨만-포드 알고리즘
    - 그래프 순회: 그래프 내의 모든 노드를 한번씩 살펴봄
        - DFS(깊이우선탐색)
            - 현재 탐색 중인 노드를 기준으로 연결된 노드를 즉시 방문
            - 재귀함수로 구현 가능

            ```python
            N = 5 # number of nodes
            
            # DFS function
            def dfs(node):
            	global adj_list, visited
            
            	# base case
            	if visited[node]: 
            		return
            	visited[node] = True
            
            	print(node, end=' ')
            
            	# recursive case
            	for adj_node in adj_list[node]:
            		dfs(adj_node)
            
            adj_list = [[] for _ in range(N)]
            visited = [False] * N
            
            # Create Adjacency List
            adj_list[0].append(1); adj_list[0].append(2)
            adj_list[1].append(4); adj_list[1].append(3)
            adj_list[2].append(3)
            adj_list[3].append(0)
            
            # Execute Depth-First Search with start node '0'
            dfs(0)
            
            ```

        - BFS (너비우선탐색)
            - 시작 노드를 기준으로 거리가 가까운 노드를 먼저 방문
            - queue 를 이용하여 구현

            ```python
            from collections import deque
            
            N = 5 # number of nodes
            
            adj_list = [[] for _ in range(N)]
            visited = [False] * N
            
            # Create Adjacency List
            adj_list[0].append(1); adj_list[0].append(2)
            adj_list[1].append(4); adj_list[1].append(3)
            adj_list[2].append(3)
            adj_list[3].append(0)
            
            # Execute Breadth-First Search with start node '0'
            q = deque()
            q.append(0)
            visited[0] = True
            
            while q:
                node = q.popleft()
                print(node, end=' ')
            
                for adj_node in adj_list[node]:
                    if visited[adj_node]:
                        continue
                    q.append(adj_node)
                    visited[adj_node] = True
             
            ```

    - 그래프 최단 경로 알고리즘
        - 가중 그래프가 아닌 경우, 경로상의 간선의 수가 적은 것 (BFS)
        - 가중 그래프의 경우 간선의 가중치 합이 작은 것
        - 음수 사이클인 경우 최단 거리가 정의 될 수 없다
        - 다익스트라 알고리즘
            - 한 노드에서 다른 모든 노드까지의 최단 경로/거리
            - 그리디한 성질을 가짐
            - 음수 간선이 존재하지 않을 떄만 사용할 수 있다
            - heap(priority queue)를 사용하여 구현 가능 (시간복잡도: O(M logN) (N: 노드의 개수, M: 간선의 개수))

            ```python
            from queue import PriorityQueue
            
            INF = int(1e12)
            N = 5
            
            adj_list = [[] for _ in range(N)]
            dist = [INF] * N
            
            # Create Adjacency List
            adj_list[0].append([1, 5]); adj_list[0].append([3, 1]);
            adj_list[1].append([2, 2])
            adj_list[2].append([4, 2])
            adj_list[3].append([1, 2]); adj_list[3].append([4, 7]);
            
            # Execute Dijkstra Algorithm with standard(start) node '0'
            pq = PriorityQueue()
            pq.put([0, 0]) # 거리, 노드
            dist[0] = 0
            
            while not pq.empty(): # pq.queue
                cur_dist, cur_node = pq.get()
            
                if cur_dist > dist[cur_node]: # 추가된 부분
                    continue
            
                for adj_node, adj_dist in adj_list[cur_node]:
                    temp_dist = cur_dist + adj_dist
                    if temp_dist < dist[adj_node]:
                        pq.put([temp_dist, adj_node])
                        dist[adj_node] = temp_dist
            
            # Print result
            print(dist)
            ```

        - 벨만-포드 알고리즘
            - 한노드에서 다른 모든 노드까지의 최단 경로/거리
            - 음수 사이클이 존재 하지 않을 때 사용 가능
            - O(NM)
        - 플로이드-워셜 알고리즘
            - 모든 노드에서 모든 노드까지의 최단 경로/거리
            - 음수 사이클이 존재하지 않을 때 사용 가능
            - O(N^3)
